🚀 **Bitcoin Technical Analysis Dashboard**
Interactive Web Dashboard built with Streamlit to analyze historical Bitcoin data up to March 6, 2026. This tool allows users to upload raw datasets and instantly generate professional-grade technical charts with industry-standard indicators.

📋 **Core Features**
Custom Data Processing: Seamlessly handles CSV uploads with automated Unix timestamp conversion.
Multi-Timeframe Analysis: Dynamically resample data from 1-minute intervals up to weekly (1min, 5min, 15min, 1H, 4H, 1D, 1W).
Interactive Candlestick Charts: Powered by Plotly for smooth zooming, panning, and precise data inspection.
Advanced Technical Indicators:
Trend: Simple Moving Averages (SMA 20 & SMA 50).
Volatility: Bollinger Bands (Upper, Middle, Lower).
Momentum: RSI 14 with overbought/oversold threshold lines.
Trend Strength: MACD (Line, Signal, and Histogram)
Date Range Filtering: Focus on specific market cycles or events using an intuitive calendar selector.

🛠️ **Built With**
Python - Core programming language.
Streamlit - Framework for the interactive web interface.
Pandas - Data manipulation and resampling.
Plotly - High-fidelity financial visualizations.
TA - For precise financial indicator calculations.

📊 **CSV Data Requirements**
To ensure the dashboard functions correctly, your CSV file should include the following columns:
Timestamp: Unix timestamp (in seconds).
Open, High, Low, Close: Price data.
Volume: Trading volume.
